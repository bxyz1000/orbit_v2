# 09 Achievement System

## Purpose
Achievements provide milestones for long-term engagement. Unlike PRs, which are numeric, Achievements are categorical milestones.

## Tiers
- **Bronze**: Entry-level actions (e.g., "First Task Completed").
- **Silver**: Consistency-based (e.g., "7 Day Streak").
- **Gold**: Volume-based (e.g., "500 Tasks Completed").
- **Orbit Platinum**: Elite level (e.g., "1 Year of Orbit").

## Categories
- **Architect**: Planning and Goal setting.
- **Warrior**: Workout and Health.
- **Monk**: Focus and Sleep.
- **Operator**: Task execution.

## Rules
- Achievements are permanent and non-revocable once earned.
- They do not add to the Daily Score (to prevent double-counting) but exist as a separate prestige layer.
