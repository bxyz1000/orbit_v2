# 05 Bonus System

## Purpose
The Bonus System is designed to reward exceptional performance and long-term consistency. It provides the "peak" scores in the unlimited Orbit scoring environment.

## Bonus Types

### 1. Beat Yesterday
- **Trigger**: Finalized Score for Today > Finalized Score for Yesterday.
- **Reward**: +25 points (Applied to Today's score during finalization).

### 2. Beat Weekly Average
- **Trigger**: Finalized Score for Today > (Sum of last 7 days / 7).
- **Reward**: +50 points.

### 3. Beat Monthly Average
- **Trigger**: Finalized Score for Today > (Sum of last 30 days / 30).
- **Reward**: +100 points.

### 4. Personal Record (Daily)
- **Trigger**: Finalized Score for Today > Current All-Time High Daily Score.
- **Reward**: +200 points.

### 5. Perfect Balance
- **Trigger**: User scores at least 50% of the target in every pillar (Tasks, Habits, Health, Focus) on a single day.
- **Reward**: +75 points.

### 6. Streak Milestones
- **7 Day**: +100
- **30 Day**: +500
- **100 Day**: +2000
- **365 Day**: +10000

## Rules
- Bonuses are calculated and applied at **Daily Finalization (11:59:59 PM)**.
- Bonuses do not stack with themselves (e.g., You can't get "Beat Yesterday" twice for the same day).
- Streak bonuses are one-time rewards per milestone achievement.
