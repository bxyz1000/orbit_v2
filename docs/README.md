# Orbit Documentation

## Overview
Orbit is a **Personal Operating System** designed for high-agency individuals who seek to "Become Better Than Yesterday." Unlike traditional productivity tools, Orbit is built on a philosophy of self-competition, balance, and consistency. 

This documentation serves as the official source of truth for the engineering and design of the Orbit platform.

## Principles
1. **Self-Competition Only**: No social leaderboards. The only opponent is the user's past self.
2. **Weighted Balance**: High scores require performance across all pillars (Tasks, Habits, Focus, Health).
3. **Consistency over Intensity**: The system rewards long-term adherence more than short-term bursts.
4. **Volume of Work**: Recognition of total output as a valid metric for growth.
