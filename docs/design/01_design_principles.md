# 01 Design Principles

## Purpose
Visual and UX guidelines for the Orbit interface.

## Principles
1. **Focus over Features**: The UI should disappear, leaving only the data and the action.
2. **Material 3 Foundation**: Leverage the latest Android design standards with a custom Orbit aesthetic.
3. **Data Density**: High-agency users need data. Use cards and grids effectively.
4. **Haptic Feedback**: Use subtle haptics to reward completion and score milestones.
5. **Dark Mode Priority**: Orbit is often used in deep-focus or night-time environments.

## Visual Language
- **Colors**: Deep blacks, vibrant accents for categories (Blue for Focus, Green for Health, Purple for Habits).
- **Typography**: Inter (Modern, readable, professional).
- **Radius**: Large, smooth corners (16pt-24pt) to create a premium, tactile feel.
